Introduction

Welcome to the introduction pages of the turtlebot project.

Here you can find out more about the project and how to use it. The turtlebot project allows you to write Python code to control the turtlebot as well as a simulation of it. Further information on the turtlebot can be viewed in the About section. Happy coding!

Commands

Basic commands

turtle.down()

Moves the turtlebot's pen down to start drawing

turtle.up()

Moves the turtlebot's pen up to stop drawing

turtle.forward(n)

Moves the turtlebot forward by n millimeters

turtle.right(n)

Turns the turtlebot clockwise by n degrees

turtle.left(n)

Turns the turtlebot anticlockwise by n degrees

print("text")

Show the word text on the turtlebot screen (under 10 characters)

Comments

Type # to comment out something. This means that it will be ignored when you press the run button

For Loops

For loops can be used to repeat commands for a finite amount of times

To repeat a command four times we can write: 

for i in range(4):
	
	do something

Connecting to the turtle

Initial connection

  1. Launch the application
  2. Plug in the turtle
  3. If the USB light is green, press the button on the turtle
  4. Wait a moment, if the turtle says "Hello", you have successfully connected

Resetting the connection

  1. Press the reset button in the top right corner
  2. Make sure the turtle is on
  3. If the USB light is green, press the button on the turtle
  4. Wait a moment, if the turtle says "Hello", you have successfully reconnected

Troubleshooting

  • Is the turtbot turned on and displaying "Turtle" on the screen?
  • Is the USB dongle plugged in?
  • Are both lights green at the top?
