Introduction

Welcome to the introduction pages of the turtlebot project.

Here you can find out more about the project and how to use it. The turtlebot project allows you to write Python code to control the turtlebot as well as a simulation of it. Further information on the turtlebot can be viewed in the About section. Happy coding!

Commands

Basic commands

turtle.down()

Moves the turtlebot's pen down to start drawing

turtle.up()

Moves the turtlebot's pen up to stop drawing

turtle.forward(n)

Moves the turtlebot forward by n millimeters

turtle.right(n)

Turns the turtlebot clockwise by n degrees

turtle.left(n)

Turns the turtlebot anticlockwise by n degrees

turtle.curve(arc length, angle)

Moves the the turtlebot forward by the specified arc length in millimeters around a curve of angle degrees. \n
turtle.curve(100, 60) produces:

turtle.message("text")

Show the word text on the turtlebot screen (under 10 characters)

Advanced coding features

Comments

Type # before what you want to be commented out. This means that it will be ignored when you press the run button

For Loops

For loops can be used to repeat commands for a finite amount of times

To repeat a command four times we can write: 

for i in range(4):
	
	do something

While Loops

While loops can be used to repeat commands until a condition is true

The basic layout is: 

while condition:
	
	do something

Caution can lead to infinite loops which will cause the turtle app to crash

Importing modules

Modules can be imported to provide additional things to do

One example of this is to use the random library to get the turtle to move a random number between 0 and 100 forwards

import random
length = random.randint(0, 100)
turtle.forward(length)

Caution can lead to infinite loops which will cause the turtle app to crash

Connecting to the turtle

Initial connection

  1. Launch the application
  2. Plug in the turtle
  3. Click on Settings on the top bar
  4. Click on Select port
  5. Select the port that the USB dongle is connected to via the dropdown
  6. If the USB light is green, press the button on the turtle
  7. Wait a moment, if the turtle says "Hello", you have successfully connected

Changing the connected port

Follow the same steps as for setting up the initial connection

Troubleshooting

  • Is the turtlebot turned on and displaying "Turtle" on the screen?
  • Is the USB dongle plugged in?
  • Have you selected a port?
  • Have you pressed the button on the turtle?
  • Are both lights green at the top?
  • If the USB light is green, press the button on the turtle
  • If the turtlebot still will not connect check that the lights on the turtle's bluetooth module and the USB dongle are both solid
    • If still flashing press both of their buttons
    • Once solid redo the connection steps

About the turtlebot

The turtlebot has been created with a mixture of 3D printing and other components including an ardunio. The chasis was designed using OpenSCAD.

The turtlebot has four main features. Two wheels that are controlled by stepper motors to move the turtlebot, a screen display and a component where a pen can be put in which can then be moved vertically by a servo. The turtlebotâ€™s firmware uses an Arduino and also has a Bluetooth component in order to receive and acknowledge commands from the PC.

The turtlebot has commands to do various actions. The movement commands have options to move in either millimetres or motor steps (with 4096 steps per rotation). Initially, the command options for moving forward, left, or right then later additional commands where added for curves.
